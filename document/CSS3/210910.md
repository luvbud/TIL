# 1. transform
## 1.1. transform (변형) 개요
 * 요소에 회전, 크기 조절, 기울이기, 이동 효과를 부여할 수 있음
 * 시각적 서식 모델의 좌표 공간을 변경함
 * 값
   - `<transform-function>`
     + 요소에 적용할 하나 이상의 CSS 변형 함수
     + 요소에 변형을 오른쪽부터 왼쪽으로 하나씩 순서대로 적용함
     + 2D, 3D 공간 내에서 요소를 회전하고, 크기를 바꾸고, 왜곡하고, 이동할 수 있음
   - **none**

## 1.2. 크기 조절
### `scale()`
 - 2D 평면에서 요소를 확대 / 축소
 - 차지하는 공간은 그대로고 요소만 사이즈가 변형됨
 > scale(sx) ➡️ sx(x축) 크기만큼 비율 유지하여 조정됨<br>
 > scale(sx, sy)
 - \<number> 값을 받음
#### `scaleX()`
 - 요소의 x축 크기 조정
#### `scaleY()`
 - 요소의 y축 크기 조정

## 1.3. 회전
### `rotate()`
 - 2D 평면의 고정점을 중심으로 요소를 회전
 > rotate(a)
 - \<angle> 값을 받음 (deg, turn, ...)
#### `rotateX()`
 - x축을 중심으로 요소 회전
#### `rotateY()`
 - y축을 중심으로 요소 회전
#### `rotateZ()`
 - z축을 중심으로 요소 회전

## 1.4. 이동
### `translate()`
 - 2D 평면에서 요소를 이동
 - \<length>, \<percentage> 값을 받음
 - 단일 값이 올 경우 ➡️ (x좌표 = 해당 값, y좌표 = 0)으로 설정됨
 - 이중 값이 올 경우 ➡️ (x좌표, y좌표)를 나타냄
#### `translateX()`
 - 요소를 x축을 따라 이동
#### `translateY()`
 - 요소를 y축을 따라 이동
#### `translateZ()`
 - 요소를 z축을 따라 이동

## 1.5. 기울이기
### `skew()`
 - 2D 평면에서 요소를 기울임
 > skew(ax) ➡️ (x좌표 = ax, y좌표 = 0)으로 설정됨<br>
 > skew(ax, ay)
 - \<angle> 값을 받음 (deg, turn, ...)
#### `skewX()`
 - 요소를 수평 방향으로 기울임
#### `skewY()`
 - 요소를 수직 방향으로 기울임

## 1.6. 기준점
### `transform-origin`
 - 요소의 변환에 대한 기준점을 설정함
 - 기본값 : 50% 50% 0 (center)

# 2. transition
## 2.1. transition (전환) 개요
 - 정해진 시간동안 요소의 속성 값을 부드럽게 변화시킬 수 있음

## 2.2. 속성
### `transition-property`
all

### `transition-duration`
 - \<time> 단위 s, ms 0.5s = 500ms


### `transition-delay`
 - 트리거 되는 시점
 - \<time>

### `transition-timing-function`
 * 값
   - ease
     + 기본값
   - ease-in
   - ease-out
   - ease-in-out
   - linear
     + 등속도
   - cubic-bezier()
   - steps()

### `transition` (shorthand)
 * **단축 속성**
 * 기본값
  - transition-delay : 0s
  - transition-duration : 0s
  - transition-property : all
  - transition-timing-function : ease

```css
/* Apply to 1 property */
/* property name | duration */
transition: margin-right 4s;

/* property name | duration | delay */
transition: margin-right 4s 1s;

/* property name | duration | easing function */
transition: margin-right 4s ease-in-out;

/* property name | duration | easing function | delay */
transition: margin-right 4s ease-in-out 1s;

/* Apply to 2 properties */
transition: margin-right 4s, color 1s;

/* Apply to all changed properties */
transition: all 0.5s ease-out;

/* Global values */
transition: inherit;
transition: initial;
transition: revert;
transition: unset;
```

# 3. 애니메이션
## 3.1. 애니메이션 개요


## 3.2. `@keyframes`
 - 개발자가 애니메이션 중간중간의 특정 지점들을 거칠 수 있는 키프레임들을 설정함
 - 애니메이션 과정의 중간 절차를 제어할 수 있게 함

## 3.3. 속성
### `animation-name`
 - 기본값 : `none`
 - 대소문자를 구별함
 - a-z, 0-9, _, - 만 사용 가능

### `animation-duration`
 - 애니메이션이 한 **사이클**(animation은 반복적으로 재생함)을 완료하는 데 걸리는 시간을 지정함
 - \<time> 값을 사용 (s, ms)
 - 음수 값 ❌

### `animation-delay`
 - 애니메이션이 시작할 시점을 지정함
 - \<time> 값을 사용
 - 음수 값 ⭕️ ➡️ 음수 값을 지정하면 애니메이션이 즉시 시작되지만 애니메이션 주기의 도중에 시작됨

### `animation-timing-function`
 * 중간 상태들의 전환을 어떤 시간 간격으로 진행할지 지정함
 * 값
   - `ease`
     + 기본값
   - `ease-in`
   - `ease-out`
   - `ease-in-out`
   - `linear`
     + 등속도 
   - `step-start`
   - `step-end`
   - `cubic-bezier()`
   - `steps()`

### `animation-iteration-count`
 * 애니메이션을 몇 번 반복할지 지정함
 * 값
   - `infinite`
     + 무한대로 재생됨 
   - \<number>
     + 기본값 : 1 (한 번만 재생됨)

### `animation-direction`
 * 애니메이션이 앞으로, 뒤로, 앞뒤로 번갈아 재생되어야하는지의 여부를 지정함
 * 값
   - `normal`
     + 기본값
     + 애니메이션이 매 사이클마다 정방향으로 재생됨 
   - `reverse`
     + 애니메이션이 매 사이클마다 역방향으로 재생됨 
   - `alternate`
     + 애니메이션이 매 사이클마다 각 주기의 방향을 뒤집음
     + 첫 번째 반복은 정방향으로 진행됨 
   - `alternate-reverse`
     + 애니메이션이 매 사이클마다 각 주기의 방향을 뒤집음
     + 첫 번째 반복은 역방향으로 진행됨 

### `animation-play-state`
 * 애니메이션을 멈추거나 다시 시작할 수 있음
 * 값
   - `running`
     + 기본값
     + 계속 재생되고 있는 상태
   - `paused`
     + 일시정지

### `animation-fill-mode`
 * 애니메이션이 실행 전과 후의 스타일을 적용하는 방법을 지정함
 * 값
   - `none`
     + 기본값
     + 애니메이션이 실행되지 않을 때 대상에 스타일을 적용하지 않음
   - `forwards`
     + 실행된 애니메이션의 마지막 keyframe 에 의해 설정된 값을 유지함
     + 
       | animation-direction | animation-iteration-count | last keyframe encountered |
       | ------------------- | ------------------------- | ------------------------- |
       | normal              | even / odd                | 100% / to                 |
       | reverse             | even / odd                | 0% / from                 |
       | alternate           | even                      | 0% / from                 |
       | alternate           | odd                       | 100% / to                 |
       | alternate-reverse   | even                      | 100% / to                 |
       | alternate-reverse   | odd                       | 0% / from                 |
   - `backwords`
     + 애니메이션은 대상에 적용되는 즉시 첫 번째 관련 keyframe 에 정의된 값을 적용함
     + animation-delay 기간 동안 이 값을 유지함
     + 
       | animation-direction         | first relevant keyframe |
       | --------------------------- | ----------------------- |
       | normal / alternate          | 0% / from               |
       | reverse / alternate-reverse | 100% / to               |
   - both
     + 애니메이션은 앞뒤 양쪽 모두의 규칙을 따르므로 애니메이션 속성이 양방향으로 확장됨

## 3.8. `animation` (shorthand)
 * 다수의 스타일을 전환하는 애니메이션을 적용함
 * 기본값
   - animation-name : none
   - animation-duration : 0s
   - animation-timing-function : ease
   - animation-delay : 0s
   - animation-iteration-count : 1
   - animation-direction : normal
   - animation-fill-mode : none
   - animation-play-state : running

```css
/* @keyframes duration | easing-function | delay |
iteration-count | direction | fill-mode | play-state | name */
animation: 3s ease-in 1s 2 reverse both paused slidein;

/* @keyframes name | duration | easing-function | delay */
animation: slidein 3s linear 1s;

/* @keyframes name | duration */
animation: slidein 3s;
```
